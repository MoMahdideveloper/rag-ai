
# هوشمند اِملاک - Smart Real Estate CRM with Gemini

**هوشمند اِملاک** یک سیستم مدیریت ارتباط با مشتری (CRM) هوشمند و پیشرفته برای مشاوران املاک است. این اپلیکیشن با بهره‌گیری از قدرت **Google Gemini API**، فرآیندهای مدیریت مشتریان، فایل کردن املاک و پیدا کردن گزینه‌های مناسب برای هر دو طرف معامله را به صورت هوشمند و خودکار انجام می‌دهد.

تمام داده‌ها (مشتریان، املاک، وظایف و کلیدهای API) به صورت امن در حافظه محلی مرورگر شما (`localStorage`) ذخیره می‌شوند و به هیچ سرور خارجی ارسال نمی‌گردند.



---

## ✨ ویژگی‌های کلیدی

- **داشبورد جامع:** نمایش کلی آمار مشتریان (متقاضیان خرید و اجاره)، وظایف باز و قراردادهای بسته شده در یک نگاه.
- **ثبت هوشمند مشتری:** با وارد کردن توضیحات مشتری به زبان طبیعی، فرم اطلاعاتی او (شامل نام، شماره تماس، و نیازمندی‌ها) به صورت خودکار توسط Gemini تکمیل می‌شود.
- **برچسب‌گذاری هوشمند:** Gemini یادداشت‌های شما را تحلیل کرده و برچسب‌های کلیدی و کاربردی (مانند "زوج جوان"، "سرمایه‌گذار"، "محیط دنج") برای هر مشتری تولید می‌کند.
- **پیدا کردن گزینه‌های ایده‌آل (Matching):**
    - **برای ملک:** برای یک ملک ثبت‌شده، بهترین مشتریان متقاضی را با درصد تطابق و دلیل پیشنهاد پیدا کنید.
    - **برای مشتری:** برای یک مشتری، مناسب‌ترین املاک موجود را بر اساس نیازمندی‌های دقیق او بیابید.
- **دستیار هوشمند (AI Copilot):** یک رابط چت برای گفتگو با هوش مصنوعی. می‌توانید سوالات خود را در مورد داده‌های CRM (مشتریان، املاک، وظایف) به زبان فارسی بپرسید و پاسخ‌های دقیق دریافت کنید.
- **مدیریت وظایف و تقویم:**
    - ایجاد، ویرایش و حذف وظایف با تعیین اولویت و تاریخ سررسید.
    - نمایش وظایف در یک تقویم ماهانه تعاملی.
- **واردات و خروجی داده:**
    - **واردات هوشمند:** فایل‌های JSON حاوی اطلاعات مشتریان یا املاک را وارد کنید تا Gemini آن‌ها را تحلیل، پاکسازی و در سیستم ثبت کند.
    - **پشتیبان‌گیری:** از تمام داده‌های خود با یک کلیک خروجی JSON بگیرید.
- **تنظیمات پیشرفته API:** امکان افزودن چندین کلید API Gemini برای جلوگیری از توقف سیستم در صورت اتمام محدودیت یک کلید.

## 🚀 راه‌اندازی و استفاده

برای استفاده از قابلیت‌های هوشمند این اپلیکیشن، به یک کلید API از **Google AI Studio** نیاز دارید.

1.  **دریافت کلید API:**
    - به [Google AI Studio](https://aistudio.google.com/) بروید.
    - با حساب گوگل خود وارد شوید.
    - روی دکمه `Get API key` کلیک کرده و یک کلید جدید بسازید (`Create API key`).
    - کلید ساخته‌شده را کپی کنید.

2.  **افزودن کلید به اپلیکیشن:**
    - در اپلیکیشن، به بخش **تنظیمات** بروید (آیکن چرخ‌دنده).
    - کلید API خود را در کادر مربوطه وارد کرده و روی دکمه "افزودن" کلیک کنید.

اکنون تمام قابلیت‌های هوش مصنوعی فعال شده و آماده استفاده است!

## ⚙️ Environment Variables

For the backend server to run correctly, you need to create a `.env` file in the `backend` directory.

```
# backend/.env

# A strong, secret key for signing authentication tokens
JWT_SECRET=your-super-secret-key-goes-here

# Your API key from SendGrid for the email feature
SENDGRID_API_KEY=SG.xxxxxxxxxxxxxxxxxxxxxx

# An email address you have verified with SendGrid
FROM_EMAIL=your-email@example.com
```

## 🛠️ تکنولوژی‌های استفاده شده

- **Frontend:** React, TypeScript
- **Styling:** Tailwind CSS
- **AI:** Google Gemini API (`@google/genai`)
- **State Management:** React Hooks (`useState`, `useEffect`, etc.)
- **Persistence:** SQLite Database via Node.js/Express Backend
